# 电动车NFC启停控制

### 介绍

* 前端：基于uniapp开发的安卓APP

* 模块程序：基于esp8266/esp12s模组以及RC522-RFID模块，使用Arduino开发

* PCB原理：[8266通过NFC控制继电器 - 嘉立创EDA开源硬件平台 (oshwhub.com)](https://oshwhub.com/dt27/wifi-switch_copy)
